# Social Media Accounts

Key individuals and organisations in the Flipper Zero ecosystem maintain social media presences where firmware releases, security research, and new tool announcements are first shared. This catalog tracks official brand accounts, firmware project accounts, prominent researchers, and tool developers across Twitter/X, Mastodon, and Bluesky. Following these accounts is the fastest way to track new Sub-GHz signal databases, CFW releases, CVE disclosures, and community events. Status reflects whether the account posted within the last 12 months as of the snapshot date.

## Legend

| Platform | Social network name |
| Last Post | `YYYY-MM` of most recent relevant post |
| Status | `active` = posted in last 12 months; `stale` = no post in 12–24 months; `archived` = account suspended/deleted |
