# CTFs, Conferences & Meetups

Flipper Zero has become a fixture at hardware security conferences and RF/NFC-focused CTF challenges. This catalog covers annual security conferences where Flipper hardware or related research has been presented (DEF CON, Black Hat, CCC, Hardwear.io), dedicated CTF events that accept Flipper as a competition tool, and community-run online contests. The `Next/Last` column records the most recent past occurrence or next scheduled date where known. Events marked `stale` either did not run in the last cycle or have not yet announced a future date.

## Legend

| Column | Values |
|--------|--------|
| Frequency | How often the event occurs |
| Next/Last | Most recent past occurrence (`YYYY-MM`) or next scheduled date |
| Type | `Conference`, `CTF`, `Contest`, `Meetup` |
| Status | `active` = recurring with recent occurrence; `stale` = skipped a cycle or unconfirmed future date |
