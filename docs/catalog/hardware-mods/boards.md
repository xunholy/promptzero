- **Name** — board or module name
- **URL** — shop, repository, or product page
- **Author** — manufacturer or maintainer
- **Stars** — approximate GitHub stars (if applicable)
- **Last Commit** — most recent update or stock date (YYYY-MM)
- **License** — hardware/firmware license
- **Status** — active/stale/archived
- **Notes** — capabilities and FAP compatibility
